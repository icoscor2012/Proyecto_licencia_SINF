# Hollow Requiem

Juego indie ficticio – Licencia GPL-3.0

## Descripción del juego

Hollow Requiem es un juego indie ficticio de acción y aventura en 2D. Combina exploración profunda, plataformas precisas y combates exigentes. El jugador recorre un mundo interconectado lleno de secretos, criaturas misteriosas y regiones de atmósferas muy distintas entre sí.

A lo largo del viaje, se encuentran personajes expresivos y memorables, cada uno con su propia historia, y una banda sonora emocional que acompaña los momentos clave del recorrido. El objetivo del jugador es desentrañar el origen de un antiguo lamento que parece resonar por todo el reino, mientras adquiere nuevas habilidades que permiten avanzar a zonas antes inaccesibles.

Este repositorio contiene el código fuente del proyecto con fines educativos.
